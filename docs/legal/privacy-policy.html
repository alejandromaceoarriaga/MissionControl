Privacy Policy

Last updated: April 9, 2026

1. Introduction

El Dato Oculto Manager ("the App", "we", "us") is an automated content management tool that connects to TikTok via the TikTok for Developers API to publish short-form video content on behalf of its operator.

2. Information We Collect

The App collects and processes only the information strictly necessary to operate:

  • TikTok OAuth tokens: Access and refresh tokens provided by TikTok's authorization flow. Stored locally on the operator's machine and never transmitted to third parties.
  • Video metadata: Video IDs, view counts, likes, comments, and shares — retrieved from the TikTok API for analytics purposes.
  • Comment text: Public comments on published videos, used solely to generate automated replies via the Claude AI API.

3. How We Use Information

  • To publish video content to TikTok on the operator's behalf.
  • To generate AI-powered replies to public comments.
  • To compute engagement analytics and display a local dashboard.

4. Data Sharing

We do not sell, rent, or share personal data with third parties. The only external API calls made are to:

  • TikTok API (open.tiktokapis.com) — for content publishing and analytics.
  • Anthropic Claude API (api.anthropic.com) — for AI script generation and comment replies.
  • Google TTS (translate.google.com) — for text-to-speech audio generation.

5. Data Retention

All data is stored locally on the operator's machine. No database or cloud storage is used. Tokens are retained until manually deleted. Video metadata is cached in a local JSON file.

6. User Rights

Users whose comments are processed have the right to request deletion of their data by contacting us at the email below. Comment data is not retained beyond the session in which it is processed.

7. Security

All credentials are stored in environment variables on the operator's local machine. The App does not expose any data to the public internet.

8. Children's Privacy

The App is not directed at children under 13. We do not knowingly collect information from children.

9. Changes to This Policy

We may update this policy. Changes will be reflected by updating the "Last updated" date above.
